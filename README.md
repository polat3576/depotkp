# Restoran Depo Takip Sistemi

Restoran çalışanlarının depodan aldığı ürünleri sisteme girip stoktan otomatik düşülmesini sağlayan, yöneticinin ürün/stok/sayım yönetimi ve tüketim raporlarını görebildiği sistem.

## Klasör Yapısı

- `server/` — Backend (Node.js + Express, Neon PostgreSQL)
- `client/` — Frontend (mobil uyumlu web panel)
- `docs/` — API dokümantasyonu, veritabanı şeması, kurulum notları

## Kurulum

Ayrıntılar için `docs/` klasörüne bakınız.
